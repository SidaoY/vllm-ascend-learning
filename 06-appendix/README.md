# 06 Appendix

附录放跨章节复用的资料：术语表、代码导航、实验模板和外部资料索引。故障排查手册已放到开发实践章节，方便和测试、CI、性能调优一起阅读。

## 文档清单

- [00-glossary.md](00-glossary.md)：术语表。
- [01-code-reading-index.md](01-code-reading-index.md)：按主题索引代码入口。
- [03-experiment-template.md](03-experiment-template.md)：实验记录模板。
- [04-references.md](04-references.md)：资料索引。

## 使用方式

阅读主线章节时，如果遇到不熟悉的缩写，先查 [术语表](00-glossary.md)。如果已经知道问题大概属于哪个模块，但不知道代码从哪里看起，查 [代码阅读索引](01-code-reading-index.md)。做性能、精度或 CI 问题复盘时，直接套用 [实验记录模板](03-experiment-template.md)，这样后续 review 和排查会轻松很多。
